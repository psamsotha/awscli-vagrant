# awscli-vagrant

Vagrant setup to use [aws-cli][cli] version 2 on Ubuntu 18.04 LTS 64-bit box

### Install

```
vagrant up
```

### Enter machine

```
vagrant ssh
```

### Run aws-cli

```bash
aws --version
aws configure
```

[cli]: https://docs.aws.amazon.com/cli/latest/userguide/cli-chap-welcome.html
